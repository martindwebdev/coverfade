Coverfade 1.0

Coverfade is a lightweight jQuery plugin that allows you display a collection of background images on your site that crossfade between each other after a specified duration of time.

Usage

Coverfade is simple to use and requires only a few short lines of code. Check out the example below or check the source of this page.

        
// an array of background images
var images = ['/img/Penguins.jpg', '/img/Desert.jpg', '/img/Chrysanthemum.jpg'];

// duration between transitions in milliseconds
var speed = 5000;

// invoke the plugin
$.coverfade(images, speed);
